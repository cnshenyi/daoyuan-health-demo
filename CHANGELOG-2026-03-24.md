# 道远健康 Demo 优化记录 (2026-03-24)

基于客户反馈，完成以下功能优化与界面改进。

---

## 1. 医生端优化

### 1.1 病历界面新增模块

**基础信息模块**

在患者详情页新增"基础信息"区域，以网格卡片形式展示以下 7 项内容：

| 字段 | 说明 |
|------|------|
| 过敏史 | 药物/食物过敏记录 |
| 既往史 | 已确诊的慢性疾病 |
| 家族史 | 直系亲属重大疾病 |
| 吸烟 | 吸烟状态及年限 |
| 饮酒 | 饮酒频率及习惯 |
| 职业 | 当前/退休职业 |
| 疫苗接种 | 已接种疫苗清单 |

**危险因素模块**

新增"危险因素"区域，记录患者健康相关的不良生活习惯和行为偏好，按风险等级分为：

- **高风险**（红色标签）：如长期吸烟、高糖饮食
- **中风险**（橙色标签）：如久坐少动、睡眠不足
- **低风险**（绿色标签）：如轻度工作压力

每条记录包含名称、详细描述和风险等级。

**长期计划功能**

在快捷操作栏新增"长期计划"按钮（与开处方、写病程、安排检查、会诊申请并列），支持：

- 管理目标选择（血糖达标、血压控制、体重管理等）
- 分阶段计划制定（阶段名称、周期、目标指标、具体措施）
- 随访频率设置（每周/每两周/每月/每季度）
- 注意事项备注

### 1.2 用药方案国际化

药物名称格式统一改为 **"中文 + 英文通用名 + (商品名)"**，方便国际医生阅读：

| 原格式 | 新格式 |
|--------|--------|
| 二甲双胍 | 二甲双胍 Metformin (Glucophage) |
| 阿托伐他汀 | 阿托伐他汀 Atorvastatin (Lipitor) |
| 氨氯地平 | 氨氯地平 Amlodipine (Norvasc) |
| 阿司匹林 | 阿司匹林 Aspirin (Bayaspirin) |
| 美托洛尔 | 美托洛尔 Metoprolol (Betaloc) |
| 华法林 | 华法林 Warfarin (Coumadin) |
| 地高辛 | 地高辛 Digoxin (Lanoxin) |

### 1.3 检查结果趋势对比

检查结果从原有的列表展示改为 **按类型分组、近三次并列对比** 的形式：

- 同类型检查（如血液检验、心电图/超声）自动归组
- 每组最多展示最近 3 次结果，按时间倒序排列
- 最新一次以左侧蓝色边框高亮
- 未出结果的检查标注"待出结果"
- 支持横向滚动浏览

## 2. 会员端优化

### 2.1 指标趋势图表

在健康档案页新增"指标趋势"标签页，为 4 项重要指标提供可视化趋势图：

| 指标 | 单位 | 正常区间 |
|------|------|----------|
| 空腹血糖 | mmol/L | 3.9 - 6.1 |
| 收缩压 | mmHg | 90 - 140 |
| 体重 | kg | 60 - 75 |
| 尿酸 | μmol/L | 208 - 428 |

图表特性：

- 基于 SVG 的轻量级折线图，展示近 6 次检测数据
- **正常区间**：绿色虚线标注上下限，绿色半透明背景填充
- **危险区间**：红色半透明背景标识超标范围
- 数据点颜色区分：绿色=正常，红色=异常
- 每个数据点标注具体数值和检测日期
- 图表下方显示图例说明
